VLSI Design Fundamentals

This course provides a comprehensive introduction to Very-Large-Scale Integration (VLSI) design, covering the fundamental principles and practical applications of designing integrated circuits. Students will learn about CMOS technology, circuit design, layout techniques, and verification methodologies. The course emphasizes hands-on experience with industry-standard tools and design projects.

Course Syllabus

Introduction to VLSI

Overview of VLSI design flow and methodologies.

CMOS Transistor Theory

Understanding the operation and characteristics of CMOS transistors.

Combinational Logic Design

Designing and optimizing combinational logic circuits.

Sequential Logic Design

Designing and optimizing sequential logic circuits.

Layout Design and Optimization

Principles of layout design, including floorplanning, placement, and routing.

Verification and Testing

Techniques for verifying and testing VLSI designs.

Required Materials

Students will need access to a computer with internet connectivity and the following software: Electronic Design Automation (EDA) tools (e.g., Cadence, Synopsys), circuit simulation software (e.g., SPICE), and a text editor. Detailed instructions for software installation and usage will be provided.
